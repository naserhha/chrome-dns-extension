🌐 DNS Configuration Manager

Smart DNS management extension for Chrome browser with multilingual support

📋 About This Extension

DNS Configuration Manager is a powerful Chrome extension that allows you to easily switch between different DNS servers. It provides a user-friendly interface with preset DNS configurations and custom DNS support, all with multilingual capabilities.

✨ Key Features

🔧 Easy DNS Switching

Quickly switch between popular DNS servers like Google DNS, Cloudflare, OpenDNS, and Quad9 with just one click.

⚙️ Custom DNS Support

Create and save your own custom DNS configurations with primary and secondary DNS servers.

🌍 Multilingual Support

Fully translated interface in English, Persian (فارسی), Arabic (العربية), and Chinese (中文).

💾 Persistent Settings

Your DNS preferences are automatically saved and restored when you reopen the extension.

📋 Quick Copy

Copy DNS addresses to clipboard for easy use in system network settings.

🔍 DNS Testing

Test DNS connections and verify that your selected DNS servers are working properly.

📥 Installation Guide

Download the Extension

Download the extension files from the GitHub repository or Chrome Web Store.

Open Chrome Extensions

Go to chrome://extensions/ in your Chrome browser.

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner.

Load Extension

Click "Load unpacked" and select the extension folder.

Start Using

Click the extension icon in your toolbar to start managing DNS settings.

⚠️ Important Information

Simulation Mode: This extension uses Chrome's proxy API to simulate DNS configuration. For actual system-wide DNS changes, you'll need to configure your system network settings manually. The extension helps you manage and copy DNS addresses for easy system configuration.

🌐 DNS配置管理器

📋 关于本扩展

DNS配置管理器是一款强大的Chrome扩展，允许您轻松切换不同的DNS服务器。它提供了用户友好的界面，内置和自定义DNS配置，并支持多语言。

✨ 主要功能

🔧 轻松切换DNS

一键快速切换Google、Cloudflare、OpenDNS、Quad9等常用DNS服务器。

⚙️ 支持自定义DNS

创建并保存您自己的主/备DNS服务器配置。

🌍 多语言支持

界面已完整翻译为中文、英语、波斯语和阿拉伯语。

💾 持久设置

您的DNS偏好会自动保存，并在重新打开扩展时恢复。

📋 快速复制

一键复制DNS地址，方便在系统网络设置中使用。

🔍 DNS测试

测试DNS连接，确保所选服务器正常工作。
