🔒 隐私政策

DNS 配置管理器扩展

📋 概述

本隐私政策描述了 DNS 配置管理器 Chrome 扩展（"我们"、"我们的"或"我们"）如何处理您的信息。我们致力于保护您的隐私并确保我们数据实践的透明度。

🛡️ 关键隐私原则

此扩展不收集、存储或传输任何用户数据。所有操作都在您的浏览器中本地进行。我们相信完全透明和最少的数据收集。

📊 数据收集

我们不收集任何个人信息。我们的扩展完全在您的浏览器内运行，不会：

  • 收集任何个人数据
  • 在外部服务器上存储信息
  • 跟踪您的浏览活动
  • 向第三方服务发送数据
  • 使用分析或跟踪工具

💾 本地存储

唯一存储的数据是您的 DNS 配置偏好，这些偏好使用 Chrome 的存储 API 本地保存在您的浏览器中。这包括：

  • 您的自定义 DNS 服务器设置
  • 语言偏好
  • 扩展状态（启用/禁用）

这些数据保留在您的设备上，永远不会传输到外部服务器。

🌐 网络通信

无外部网络请求：我们的扩展不进行任何 API 调用、fetch 请求或与外部服务器通信。所有功能都在您的浏览器内本地运行。

🔐 权限

我们的扩展需要最少的权限：

  • proxy：配置 DNS 设置
  • storage：本地保存您的偏好
  • http://*/* 和 https://*/*：访问网站（DNS 功能的标准）

我们不请求不必要的权限，如访问标签页、书签或浏览历史记录。

🔍 代码透明度

我们的扩展代码：

  • 完全可读且未压缩
  • 开源且透明
  • 无混淆或加密
  • 可供公众审查

🔄 更新

本隐私政策可能会偶尔更新。任何重要变更都将通过扩展的更新过程进行沟通。在变更后继续使用扩展即表示接受更新后的政策。
