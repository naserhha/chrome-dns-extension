🔒 سياسة الخصوصية

إضافة مدير إعدادات DNS

📋 نظرة عامة

تصف هذه سياسة الخصوصية كيفية تعامل إضافة مدير إعدادات DNS لمتصفح Chrome ("نحن"، "لنا"، أو "نحن") مع معلوماتك. نحن ملتزمون بحماية خصوصيتك وضمان الشفافية في ممارساتنا المتعلقة بالبيانات.

🛡️ مبادئ الخصوصية الأساسية

لا تجمع هذه الإضافة أو تخزن أو ترسل أي بيانات للمستخدم. جميع العمليات تتم محلياً في متصفحك. نحن نؤمن بالشفافية الكاملة وجمع البيانات الأدنى.

📊 جمع البيانات

نحن لا نجمع أي معلومات شخصية. تعمل إضافتنا بالكامل داخل متصفحك ولا:

  • تجمع أي بيانات شخصية
  • تخزن معلومات على خوادم خارجية
  • تتعقب نشاط التصفح الخاص بك
  • ترسل بيانات إلى خدمات الطرف الثالث
  • تستخدم أدوات تحليلية أو تعقب

💾 التخزين المحلي

البيانات الوحيدة المخزنة هي تفضيلات إعدادات DNS الخاصة بك، والتي يتم حفظها محلياً في متصفحك باستخدام واجهة برمجة التطبيقات للتخزين في Chrome. وهذا يشمل:

  • إعدادات خادم DNS المخصصة الخاصة بك
  • تفضيلات اللغة
  • حالة الإضافة (مفعلة/معطلة)

تبقى هذه البيانات على جهازك ولا يتم إرسالها أبداً إلى خوادم خارجية.

🌐 الاتصال بالشبكة

لا توجد طلبات شبكة خارجية: لا تقوم إضافتنا بأي مكالمات API أو طلبات fetch أو التواصل مع خوادم خارجية. جميع الوظائف تعمل محلياً داخل متصفحك.

🔐 الأذونات

تتطلب إضافتنا أذونات محدودة:

  • proxy: لتكوين إعدادات DNS
  • storage: لحفظ تفضيلاتك محلياً
  • http://*/* و https://*/*: للوصول إلى المواقع الإلكترونية (معيار لوظائف DNS)

نحن لا نطلب أذونات غير ضرورية مثل الوصول إلى علامات التبويب أو الإشارات المرجعية أو سجل التصفح.

🔍 شفافية الكود

كود إضافتنا:

  • قابل للقراءة بالكامل وغير مضغوط
  • مفتوح المصدر وشفاف
  • خالي من الغموض أو التشفير
  • متاح للمراجعة العامة

🔄 التحديثات

قد يتم تحديث سياسة الخصوصية هذه أحياناً. سيتم إبلاغ أي تغييرات مهمة من خلال عملية تحديث الإضافة. الاستمرار في استخدام الإضافة بعد التغييرات يعني قبول السياسة المحدثة.
