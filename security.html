🔒 Security & Privacy Transparent & Safe

🇺🇸 English

✅ Complete Security Checklist

  • proxy: For changing DNS settings
  • storage: For storing user settings
  • http://*/* and https://*/*: For accessing websites

🚫 Unnecessary Permissions Removed

  • activeTab, tabs, bookmarks, history: Removed

🛡️ Security Features

  • No data collection or transmission
  • All settings stored only in your browser
  • Open source & fully transparent
  • No tracking, analytics, or extra cookies
